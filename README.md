# Домашнее задание

**Цель:**
Освоить использование Celery для выполнения фоновых задач и настройку Redis как брокера задач.

**Результат:**
Приложение интернет-магазина с фоновыми задачами (например, отправка уведомлений при добавлении новых товаров.)


## что реализовано
- Настроил Celery и Redis;
- Реализовал фоновую задачу для логирования информации о добавлении нового товара:
    - Задача выводит сообщение в консоль (название нового товара и время создания);
    - Убедился, что задачи корректно ставятся в очередь и выполняются.

## как все это запустить. В консоли ввести команду для:
- make setup — настройка окружения.
- make redis — запуск Redis.
- make worker — запуск Celery‑воркера (в отдельном терминале).
- make server — запуск Django‑сервера (в другом терминале).
- make clean — очистка проекта.
- задачу также можно чекнуть в модели **django_celery_results_taskresult**

- по адресу http://127.0.0.1:8000/ можно "подергать ручки"
- по адресу http://127.0.0.1:8000/admin/ (логин: s@s.ru пароль: 1) можно поковырять админку
- командой: **python manage.py gen_data** можно дополнить модели случайными данными
- командой: **python manage.py loaddata example_datas.json** можно загрузить в модели эталонные данные
- как надоест выйти остановить сервер комбинацией **CTRL-BREAK**
- потушить раба-селери **CTRL-BREAK**
- потушить докер контейнер с редисом **docker stop redis-server**
- можно запустить все тесты командой **pytest -vv**
